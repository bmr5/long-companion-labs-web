# Long Companion Labs

Science-backed pet longevity through veterinary telehealth and compounded peptide therapy.

**Website:** [longcompanionlabs.com](https://longcompanionlabs.com)

## About

Long Companion Labs connects pet owners with licensed veterinarians via telehealth for personalized peptide therapy prescriptions. Compounded medications are shipped directly to your door from licensed pharmacies.

This is a pre-launch landing page focused on email/waitlist capture.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v4
- **Email Capture:** Supabase
- **Analytics:** PostHog
- **Hosting:** Vercel

## Getting Started

```bash
pnpm install
pnpm dev
```

## Environment Variables

```
NEXT_PUBLIC_SUPABASE_URL=

NEXT_PUBLIC_POSTHOG_KEY=

```
